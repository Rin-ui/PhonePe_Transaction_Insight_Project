 **1\. Aggregated Data**

This section provides **summary statistics** of transactions or user behavior, broken down by time and category.

**Purpose:**

Gives an overall picture of usage or transactions, like:

* *   Total UPI transactions
* *   Total recharge transactions
* *   Total app opens

**Important Columns:**

* *   category: Type of service (Recharge, UPI, etc.)
* *   payment\_type: Mode (e.g., UPI, Cards, Wallet)
* *   transaction\_count: Number of transactions
* *   transaction\_amount: Value of those transactions
* *   year, quarter\_file: Time range
* *   from\_date, to\_date: Start and end of data
* *   level: Scope (e.g., Country or State)
* *   state\_name: "null" here, means it’s at Country level

**Example Usage:**

Total number of UPI payments in Q2 of 2021 at the national level.

* * *

**2\. Map Data**

This provides **location-based metrics** useful for plotting visual maps — like heatmaps.

**Purpose:**

To show data **geospatially** — like intensity of usage in different regions or pin codes.

**Important Columns:**

* *   latitude, longitude: Geolocation coordinates
* *   metric: The actual value being mapped (could be transaction volume or count)
* *   label: Name of the place/region (e.g., district name)
* *   subfolder: Indicates whether it's for transaction or user type
* *   year, quarter\_file: Time range

**Example Usage:**

Plot a heatmap of most active districts in terms of payment activity in Q3 2022.

* * *

 **3\. Top Data**

This gives **ranked/top entities** — like most active districts, top apps, top brands, etc.

**Purpose:**

To show the **top N** entities by usage, transactions, etc.

**Important Columns:**

* *   entity\_type: What’s being ranked (e.g., district, pincode, brand, app)
* *   transaction\_count, transaction\_amount: Ranking metrics
* *   metric\_type: Type of metric used
* *   entity\_name: Name of the entity (e.g., "Samsung" or "Bangalore Urban")
* *   year, quarter\_file

**Note:**

If the data is "null" or "Error" for some fields, it means the JSON file had no valid or available data for that specific record.

**Example Usage:**

Top 10 districts by transaction amount in Q1 of 2023.
